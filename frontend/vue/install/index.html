    Vue Project Init¶

    Vue 프로젝트를 구성하기 위한 개발환경 구성 기록

    초기 설치 구성¶

    Yarn 과 Vue-cli 설치

    Yarn¶

    Yarn을 설치하고 프로젝트에 Yarn Berry 를 구성

    Quote

    Yarn Berry에 대한 자세한 내용은 토스 테크사이트에서 참고 https://toss.tech/article/node-modules-and-yarn-berry

    yarn 설치
    npm install -g yarn

    vue-cli¶

    vue-cli 설치
    yarn global add @vue/cli

## 혹은 업데이트할 경우
yarn global upgrade --latest @vue/cli

    vue create¶

    vue create . (해당 폴더에 vue 프로젝트 구성)
    vue create .

    대화형 스타일로 필요한 구성을 선택하여 프로젝트를 구성함
